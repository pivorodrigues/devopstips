# Ansible Basics

*Based on Whizlabs Course*

### Ansible Language Basics

- **Playbooks** contain **plays**

- **Plays** contain **tasks**

- **Tasks** call **modules**

- **Tasks** run **sequentially**

- **Handlers** are triggered by **tasks**, and are run once, at the end of plays
